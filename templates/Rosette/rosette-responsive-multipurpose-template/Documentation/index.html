Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside rosette folder.

Pages

    • index.html is for Default Home page
    • index-two.html for Home page version two
    • index-three.html for Home page version three
    • blog-fullwidth.html is for Blog full width page
    • blog-list-style.html is for Blog list style page
    • blog-right-sidebar.html is for Blog with Right sidebar page
    • product-left-sidebar.html is for Product left sidebar page
    • product-right-sidebar.html is for Product right sidebar page
    • product-without-sidebar.html is for Product without sidebar page
    • shop-left-sidebar.html is for shop left sidebar page
    • shop-right-sidebar.html is for shop right sidebar page
    • cart.html is for cart page
    • checkout.html is for checkout page
    • order-complete.html is for order complete page
    • contact.html is for Contact page
